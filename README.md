# 🐸 Portfolio - Karen Proust

Personal portfolio built from scratch as part of my web development journey at Hetic.

## Live

https://karenp0145.github.io/FirstPortfolio/

## Features

- Smooth single-page navigation with dot sidebar
- Dark mode (saved in localStorage)
- Responsive design with mobile burger menu
- Sections: Hero, About, Skills, Projects, Contact
- Downloadable CV & recommendation letter

## Built With

- HTML5 / CSS3 (modular stylesheets)
- Vanilla JavaScript
- Google Fonts — Space Mono & Outfit
- No framework, no library

## Structure

```
portfolio/
├── assets/
│   ├── styles/        # CSS files by section
│   ├── js/            # script.js
│   ├── img/           # images
│   └── documents/     # CV & recommendation letter
└── index.html
```
